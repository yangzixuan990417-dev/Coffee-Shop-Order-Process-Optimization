# Coffee Shop Order Process Optimization

## 项目介绍
这是一个信息系统（IS）项目。我分析了咖啡店点单流程的问题，并提出了优化方案。

## As-Is 当前流程
使用Draw.io绘制
![As-Is Flowchart](As-Is-Flow.png)

## 发现的问题
- 手动点单容易出错
- 库存不实时更新，导致缺货

## To-Be 改进思路
计划下一步绘制To-Be流程图，并使用Power BI制作销售数据仪表盘
引入POS系统 + 实时库存管理，减少等待时间。

## 用到的工具
- Draw.io (流程图)
- Power BI (数据可视化，待补充)

## 学习收获
学会了业务流程分析和可视化表达以及理解了信息系统如何优化商业流程。
