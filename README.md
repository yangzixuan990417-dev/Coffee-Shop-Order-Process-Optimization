# Coffee Shop Order Process Optimization

## 项目介绍
这是一个信息系统（IS）课程项目。我分析了咖啡店点单流程的问题，并提出了优化方案。

## As-Is 当前流程
![As-Is Flowchart](AS-Is-Coffee-Order-Buiness-Process-Flow.png)

## 发现的问题
- 手动点单容易出错
- 库存不实时更新，导致缺货

## To-Be 改进思路
引入POS系统 + 实时库存管理，减少等待时间。

## 用到的工具
- Draw.io (流程图)
- Power BI (数据可视化，待补充)

## 学习收获
学会了业务流程分析和可视化表达。
